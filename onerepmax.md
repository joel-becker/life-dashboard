### One-rep-max metric

In order to put my exercises on to a comparable scale within-set, I estimate one-rep-max (1RM) using Epley's equation:

```
1RM = weight * (1 + (reps / x)).
```

Typically `x` is set to 30, but I found that `x = 25` better tracks my progress.

### Measurement idiosyncracies

Some things to be aware of in the numbers:
- I always include only the weight held in my hands as I am performing the movement, so one-handed exercises involve ~1/2 as much weight as they might otherwise.
- I had a deadlift injury in ~October 2020, which explains some of the performance changes.
