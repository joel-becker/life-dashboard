### Metric explanation

In order to put my exercises on to a comparable scale within-set, I estimate one-rep-max (1RM) using Epley's equation:

```
1RM = weight * (1 + (reps / x)).
```

Typically `x` is set to 30, but I found that `x = 25` better tracks my progress.

### Data displayed

I include only exercises for which I have registered at least 40 sets, with the most recent set occurring within the last 6 months. Within these exercises, I only plot sets which have a 1RM >2/3rds that of my first ever set on that exercise (to prevent drop-sets from polluting the charts).
