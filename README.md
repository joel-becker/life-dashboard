# life-dashboard

Shiny app visualising data from around my life.

## How to use

The data is prepared for visualisation using `tables_exporter.R`, which uses `tables_helper.R` to wrangle the data given the formats described below.

### Exercise data

Exercise data is recorded using the "Strong" app.

### Diet/health data

Diet data is recorded in "MyFitnessPal". However, in order to have other health-related data imported simultaneously, the diet data is exported using Apple's "Health" app.

### Mental health data

Mental health data is recorded in the "eMoods" app.
